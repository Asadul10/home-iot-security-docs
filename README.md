# Home IoT Security Documentation

Implementation documentation for a local-first home IoT security architecture with VLAN segmentation, MQTT over TLS/mTLS, FastAPI, Next.js, and local network security controls.

## Documents

This repository contains the following supporting implementation documents:

- `docs/Proxmox_pfSense_Network_Deployment_Report.pdf`  
  Proxmox, pfSense, Ubuntu DNS, Tailscale, and VLAN network deployment documentation.

- `docs/SmartHome_Application_Implementation_Report.pdf`  
  Smart home IoT application implementation report covering TLS/mTLS, MQTT, Docker, FastAPI, Next.js, PostgreSQL/TimescaleDB, anomaly detection, and dashboard features.
